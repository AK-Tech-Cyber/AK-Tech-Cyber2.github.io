Présentation

Bonjour et bienvenue sur mon espace personnel. Ce site internet a été réalisé dans le cadre de mon projet d'évaluation en Sciences Numériques et Technologie (SNT). Vous y découvrirez mes réalisations scolaires ainsi que mes centres d'intérêt.

Commentaires pour la grille d'évaluation

Afin de faciliter la correction, voici où se trouvent les différents critères d'évaluation demandés :

  • 4 pages Web : Accueil, Productions scolaires, Mes passions, Mes vidéos.
  • Liens relatifs : Utilisés dans la barre de navigation.
  • Liens absolus : "Mon site de rêve" dans le menu et le wiki Backrooms.
  • 5 balises différentes : Utilisation de h1, h2, p, img, iframe, ul, div, nav.
  • Plusieurs couleurs et polices : Fichier CSS avec noir, marron, rouge brique, et polices Arial/Georgia.
  • Médias : Images, Padlet interactif, vidéos YouTube.
  • Structure et Code : J'ai également veillé à bien indenter mon code pour qu'il soit lisible, en utilisant des balises sémantiques (header, main, footer) pour organiser ma page de façon logique et professionnelle.
